# Archive Folder Policy

This folder is for docs that are no longer operationally correct but worth keeping for history.

Rules:

1. Move stale docs here instead of deleting by default.
2. Add `Status: archive` to moved files.
3. Keep active runbooks/plans out of this folder.
4. Do not reference archived docs as implementation source of truth.
